# Codex形态比较

| 形态 | 适合场景 | 关键优势 |
| --- | --- | --- |
| App | 桌面协作、并行任务、可视化审查 | 本地项目、worktree、内置终端、Git 工具 |
| CLI | 终端优先、本地自动化 | 交互式和非交互式任务都方便 |
| IDE 扩展 | 代码编辑器内协作 | 贴近当前文件、选区和编辑器上下文 |
| Cloud | 长任务、并行探索 | 不占用本机环境 |
| GitHub / Slack 集成 | 团队协作 | 围绕 PR、审查和任务分派协作 |

## 选择建议
日常本地开发优先 CLI 或 IDE；多任务审查可用 App；长任务可用 Cloud；团队流程可用 GitHub / Slack 集成。
