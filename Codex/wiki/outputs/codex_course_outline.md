# Codex 课程第一版大纲

## 课程定位

这是一套面向中文课堂的 Codex 实战课程。课程不只讲界面入口，而是训练学员理解 Codex 的工作方式、权限边界、项目规则和可复用工作流。

## 适合人群

- 想把 Codex 用到真实工作流的新手。
- 需要给团队做 AI 编程或 AI 办公培训的讲师。
- 已经会用 ChatGPT，但还不熟悉 Codex 本地文件、命令、权限和项目规则的人。

## 第一版主线

| 课时 | 标题 | 核心目标 | 课堂产物 |
|---|---|---|---|
| L00 | 课程总览 | 知道课程目标和知识库方法 | 学习路线图 |
| L01 | Codex 入口地图 | 能按任务选择 App、CLI、IDE、Cloud、Mobile | 入口选择表 |
| L02 | 完成第一个任务 | 能让 Codex 完成一个小而可验证的任务 | 第一个任务 demo |
| L03 | 权限、沙盒与审批 | 能安全判断是否放行 Codex 操作 | 安全护栏图 |
| L04 | AGENTS.md 项目规则 | 能为项目写最小可用规则 | AGENTS.md 模板 |

## 课程节奏

- 讲概念：每个概念只讲和现场任务相关的部分。
- 做演示：每节课绑定一个可执行 demo。
- 练判断：让学员练“什么时候该让 Codex 做、什么时候该拒绝”。
- 留产物：每节课结束时都有一个可复用模板或清单。

## 关键提醒

- 产品入口、套餐、模型、插件和权限界面可能变化，正式授课前要核验。
- 第三方 API 和社区工具作为经验补充，不进入官方主线。
- 课堂默认使用低风险、本地、可验证任务。
