# 工具 - Codex

## 工具定位

Codex 是面向真实项目的 AI 编程与任务执行助手，可用于阅读代码、修改文件、运行命令、调试测试、整理文档和构建工作流。

## 适合场景

- 代码库理解
- Bug 修复
- 测试补充
- 文档整理
- Obsidian 知识库搭建
- 课程案例制作

## 不适合场景

- 没有上下文的泛泛问答
- 需要无审查执行高风险操作的任务
- 用户不愿提供目标、约束或验收标准的任务

## 核心功能

| 功能 | 说明 | 使用建议 |
|---|---|---|
| 读项目 | 搜索文件、理解结构 | 先让它扫描目录和关键文件 |
| 改代码 | 直接编辑工作区文件 | 给出明确目标和验收标准 |
| 跑命令 | 执行测试、构建、检查 | 让它报告失败原因和下一步 |
| 工作流 | 结合模板、SOP、工具 | 固化成可复用提示词和案例 |
