# 概念 - Agentic Coding

## 定义

Agentic Coding 指让 AI 编程助手在明确目标、权限和反馈循环下，主动阅读项目、制定步骤、修改文件、运行验证并迭代完成任务。

## 它解决什么问题

把 AI 从“回答代码问题”提升为“协作完成工程任务”。

## 核心机制

- 项目上下文读取
- 任务分解
- 工具调用
- 文件编辑
- 测试验证
- 根据反馈继续迭代
