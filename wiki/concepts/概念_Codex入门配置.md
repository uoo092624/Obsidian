# Codex入门配置

## 定义
Codex 入门配置是从“安装并能安全地让 Codex 处理本地任务”所需的一组基础动作。

## 步骤
1. 从官方入口下载安装 Codex。
2. 使用 ChatGPT 账号或 API key 登录。
3. 熟悉界面、任务框和结果区域。
4. 查看上下文状态，并在需要时手动压缩上下文。
5. 查看额度状态，理解标准/快速速度模式的消耗差异。
6. 按任务难度选择模型与智能程度。
7. 新建项目或选择本地文件夹，并设置合适权限。

## 风险与边界
- 项目权限应尽量从默认权限或自动审查开始，不要轻易开启完全访问。
- 模型速度和智能程度会影响额度消耗，复杂任务再提高配置。
