## 来源信息
- 原始文件：[[raw/Codex全解【视频文档】.md]]

- 主题：[[wiki/entities/实体_Codex|Codex]] 的完整上手与进阶使用流程。

## 核心要点
- 准备工作包括下载安装、登录方式和界面熟悉；登录可用 ChatGPT 账号或 API key。
- 基础操作包括上下文查看与压缩、额度状态查看、模型/速度/智能程度选择。
- 本地文件读写部分强调项目文件夹选择、权限设置和项目内并行任务。
- 命令行使用建议在“自动审查”模式下进行，并指出聊天任务框内文件不可直接编辑时可借助 Cursor 等 Agent IDE。
- 持久记忆分为全局级长期记忆、项目级 `AGENTS.md`、自动记忆三类。
- 大型项目可使用计划模式，从想法澄清、计划生成、执行、fork 分支、注释反馈到项目级 AGENTS.md 沉淀形成闭环。
- 插件生态包括 Browser、Computer Use、Chrome、GitHub、OpenAI Developers、Vercel、Netlify、Sentry、Remotion、HyperFrames 等。
- Skills 与 CLI 可通过链接交给 Codex 自动部署；示例包括 Find-Skill、Frontend-Design、humanizer-zh、飞书 CLI、OpenCLI、GitHub CLI、gemini-CLI。
- MCP 部分介绍位置、安装、授权与扩展学习资源；自动化任务可通过自动化面板或自然语言创建。
