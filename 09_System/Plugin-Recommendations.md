# Plugin-Recommendations

## 已可直接使用

当前结构不依赖插件也能使用：文件夹、双链、标签、front matter、模板文件都已经就绪。

## 推荐安装

### Dataview

用途：让 [[Dashboard]]、[[AI 总地图]] 和 [[课程创作地图]] 中的自动索引表格生效。

适合查询：

- 待处理资料
- 可复用知识卡
- 课程素材
- 最近输出

### Omnisearch

用途：增强全文搜索，适合本地 AI 知识库中大量 Markdown、网页摘录和课程资料检索。

### Templater

用途：如果后续希望模板自动填充更复杂的日期、标题、移动路径和变量，可以在 Obsidian 自带模板之外启用它。

## 已安装插件

你当前已有：

- Obsidian Git
- Excalidraw
- Real Claudian

## 使用建议

先用当前结构运行一周，再决定是否安装 Dataview。不要一开始安装太多插件，知识库稳定性比功能数量更重要。
