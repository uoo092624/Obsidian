# 输出物说明

`wiki/outputs/` 用来放最终可交付或半成品：

- 课程大纲
- 每节课讲义
- Marp 幻灯片
- PDF-ready Markdown
- 演示脚本合集
- 图片说明表
- 学员练习包

## 命名建议

- `codex_course_outline.md`
- `codex_marp_deck.md`
- `codex_demo_scripts.md`
- `codex_exercise_pack.md`
- `lesson_L01_handout.md`

## 输出前检查

- 是否有来源引用。
- 是否有待核验信息。
- 是否能直接课堂讲。
- 是否有演示步骤和失败恢复路径。
- 是否需要图片或截图。
