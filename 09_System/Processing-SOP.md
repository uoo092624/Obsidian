# Processing-SOP

## 每日 10 分钟

1. 把零散记录放入 `00_Inbox`。
2. 给新资料添加最小 front matter。
3. 删除明显无用或重复资料。
4. 写清楚 `next_action`。

## 每周 60-90 分钟

1. 处理 `00_Inbox`。
2. 重要原始资料移动到 `02_Sources`。
3. 生成或补全资料卡。
4. 拆出概念卡、工具卡、案例卡、提示词卡。
5. 更新对应主题地图。
6. 标记可用于课程、文章或项目方案的素材。

## 每月 2-3 小时

1. 检查标签是否膨胀。
2. 合并重复主题和重复卡片。
3. 更新 [[AI 总地图]]。
4. 归档过时工具资料。
5. 选出下月课程或文章选题。

## 知识加工链路

原始资料 -> 资料卡 -> 精读笔记 -> 主题卡片 -> 输出稿件。
