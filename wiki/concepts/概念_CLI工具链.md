# CLI工具链

## 定义
CLI 工具链是 Codex 可以通过命令行安装、配置和调用的一组外部工具，用来把不同平台能力带入本地开发和自动化流程。

## 来源中的推荐 CLI
| CLI | 功能 |
| --- | --- |
| 飞书 CLI | 覆盖消息、文档、多维表格、表格、幻灯片、日历、邮箱、任务、会议等业务域 |
| OpenCLI | 通用命令行中心与 AI 原生运行平台 |
| GitHub CLI | 在终端中处理 PR、Issue 等 GitHub 工作流 |
| gemini-CLI | 从终端访问 Gemini 模型能力 |

## 使用方式
可以把 CLI 的下载链接或安装目标直接交给 Codex，由 Codex 检查环境、安装、配置并引导授权。
