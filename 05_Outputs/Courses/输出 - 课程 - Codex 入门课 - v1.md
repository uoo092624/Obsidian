# 课程 - Codex 入门课

## 课程定位

帮助学习者掌握 Codex 的基础使用、任务表达、项目协作、文件修改、命令验证和安全边界。

## 学完之后能够做到

1. 用 Codex 理解一个本地项目。
2. 让 Codex 安全地修改文件并验证结果。
3. 把常见任务沉淀成自己的 AI 编程工作流。
